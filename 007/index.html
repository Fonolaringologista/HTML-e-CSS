Texto marcado

Podemos criar um texto marcado com a tag MARK.

O estilo da tag MARK pode ser definido na área HEAD da estrutura.

Texto deletado

Tag usada para indicar que o texto pode ser lido, mas não deve ser considerado.

Texto inserido

Tag que indica que o texto foi adicionado posteriormente.

Existe tambem a tag U para sublinhar um texto (nao semantica).

Texto sobrescrito e subscrito

Tags utilizadas para escrever textos como expoentes ou como indices.
